Al enviar una petición POST a la dirección http://localhost:3000/registro incluyendo el objeto JSON con las propiedades "nombre" y "mensaje", el servidor responde de manera exitosa devolviendo un estado HTTP 200 y un objeto JSON de confirmación. Esta respuesta del servidor contiene la propiedad '"estado": "Datos recibidos"', seguida exactamente de los mismos datos estructurados que el cliente envió originalmente. Esto demuestra que la configuración de express.json() funciona correctamente y el servidor es capaz de leer e interpretar el cuerpo de la petición.
